# Assignment D4-V7: Search and Rescue – Victim Health and Treatment Constraints
The following assignment will look at how to plan actions to treat victims depending on the severity and diagnosis of their injury. 

## PDDL
When solving the task using the basic PDDL model the health of the victims are described by discrete levels. These span from `critical` -> `serious` -> `moderate` -> `stable`. Once the victim is stable it is safe to transport them to an evacuation point (i.e. hospital).
